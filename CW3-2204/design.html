(m)

I use TWO kinds of position (And I think they are quite meaningful):

  1. I use RELATIVE POSITION in all blocks (with id selector) and set all value to 0 so that I can give absolute position to everything inside the blocks.
  2. I use ABSOLUTE POSITION for menu items' headings (with pseudo class selector) so that their position will not change menu items are shown.

(n)

I use FOUR kinds of CSS3 techniques:

  1. Every block's (excluding header and footer) border is designed with ROUND CORNER.
  2. I use GRADIENT to design the background color of the header (linear change from orange to yellow).
  3. I use TRANSITION of background color of information and order block. Whenever they are hovered, the background color will change from white to some kind of light green
  4. I use ANIMATION in the promotion block to highlight it so that people are more likely to be attracted to it.

(o)

My styles are grouped as follows

  • All of my styles are link style.
  • I created 4 stylesheets: default.css, homeDesign.css, orderDesign.css, and orderPrint.css
  • defualt.css is in the upper level folder of all HTML files. It controls the styles of header and footer.
  • homeDesign.css is grouped together with home.html. It controls the styles (information block and promotion block) of home page.
  • orderDesign.css is grouped together with order.html. It controls the styles (menu block and order table block) of order page.
  • orderPrint.css is grouped together with order.html. It shows the style when this page is printed.

CW3 event handlers:

A. Home Page

  • (HTML event handler) To show a random promotion text on page load, I declared a function promote() and set the event handler onload in the body tag.
  • (HTML event handler) To change the video when the playing is over, I declared a function changeVideo() and set the event handler onended in the corresponding video tag.
  • (HTML event handler) To check input of the reservation form, I declared a function checkForm() and set the event handler onclick in the submission button.
  • (JavaScript event handler) To cancel the event of clearing forms, I declared a function checkCancel() and set the event handler onclick in JavaScript codes by selecting the button in DOM

B. Home Page

  • (HTML event handler) To create menu items by javascript corresponding to different buttons and the default setting,
    on page load, I set event handler onload to call the function displayImages("sushi") as default the displayed menu pages. I also set eventhandlers onclick for all three menu titles to display different items.
  • (JavaScript event handler) To add tablerows for each "add" button, I use DOM to select dynamic HTML elements created by JS and set onclick event handlers to call the function addOrder().
  • Other two functions recal() and undo() are not called by event handlers. They are invoked simply inside other functions automatically.
